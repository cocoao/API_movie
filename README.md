### API를 이용한 영화 검색 사이트
> 영화 API를 받아와서 검색 기능과 최신작과 장르별로 제공<br>
> 메인페이지에서 영화제목 검색시 네이버 API를 받아와 
> 최신 / 장르를 탭을 이용하여 심플하게 구성<br>
> 최신작과 장르별 영화 클릭 시 영화 세부 정보를 yts API로 받아와 출력
<br>
